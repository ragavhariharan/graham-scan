What is Graham Scan?

Graham Scan is an algorithm that finds the convex hull — the smallest convex polygon that contains all the given points. Think of it as stretching a rubber band around all the points.

Graham Scan Algorithm

function grahamScan(points):
  // Step 1: Find lowest point
  pivot ← point with min y
    (leftmost if tie)

  // Step 2: Sort by polar angle
  sort points by angle w.r.t pivot

  // Step 3: Process each point
  stack ← [pivot, points[1]]

  for i ← 2 to n-1:
    while ccw(top2, top, p[i]) ≤ 0:
      stack.pop()
    stack.push(p[i])

  return stack // convex hull

Orientation (CCW) Test

function ccw(A, B, C):
  cross = (B.x-A.x)*(C.y-A.y)
        - (B.y-A.y)*(C.x-A.x)
  if cross > 0: return LEFT TURN
  if cross < 0: return RIGHT TURN
  return COLLINEAR

🎯 Key Concepts

Convex Hull

The smallest convex polygon enclosing all points. Like stretching a rubber band around nails on a board.

Polar Angle

The angle formed between the positive x-axis and the line from the pivot to each point. Used to sort points counter-clockwise.

Cross Product

For vectors AB and AC: (B.x−A.x)·(C.y−A.y) − (B.y−A.y)·(C.x−A.x). Positive = left turn, negative = right turn.

Left Turn vs Right Turn

Left turn (CCW): The point is on the correct side — keep it on the hull.
Right turn (CW): The previous point makes the hull concave — remove it.

📖 Algorithm Steps

  1. Find the pivot: The bottommost (lowest y) point. If there's a tie, pick the leftmost.
  2. Sort points: Order all other points by polar angle relative to the pivot.
  3. Initialize stack: Push the pivot and the first sorted point.
  4. Process each point: For each remaining point, check orientation with the top two stack points.
  5. Adjust hull: Pop from the stack while a right turn (clockwise) is detected.
  6. Complete: The stack contains the convex hull vertices in counter-clockwise order.
